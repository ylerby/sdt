# Лабораторная работа по теории разработки программного обеспечения
### Для запуска необходимо:
* Выполнить команду:
  ```
  go mod tidy
  ```
* Перейти в /backend/cmd
* Выполнить команду:
  ```
  go run main.go 
  ```
