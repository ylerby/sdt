# Лабораторная работа по теории разработки программного обеспечения

### Для запуска frontend`a необходимо:

```
cd frontend
npm i
npm start

link: http://localhost:3000
```

### Для запуска backend`a необходимо:

- Выполнить команду:
  ```
  go mod tidy
  ```
- Перейти в /backend/cmd
- Выполнить команду:
  ```
  go run main.go
  ```
